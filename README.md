# RPM build files for the efitools package  
Tested using RHEL 7.6 x86_64. The efitools package provides many useful tools for managing efi systems.

## Dependencies
- gnu-efi-devel
- sbsigntools
- binutils-devel
- openssl-devel
- rpm-build
- openssl

## Building
- Clone this repository
- Ensure all dependencies are met
- run ./build.sh
